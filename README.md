# Humdrum's List of Places

An example app using Express and Sequelize with Jade views.

__on command line run:__
to instal - `npm install`

to serve app - `./bin/www`

##### Includes Models:
* __Places__
  have many links
* __Links__
    belong to one place

##### Has routes for:
* Creating/Deleting places
* Creating/Deleting links
